## Environment Variables

Postman:

- use environment variables e.g. `{{host}}` instead of `localhost:4567`
- `GET httt://{{host}}/lists`

Insomnia:

- use environment variables e.g. `{'host':'localhost:4567'}`
- just type `host` for auto complete in URL editing
